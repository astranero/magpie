# Contributing to Magpie

Thanks for your interest. This project is a side project built out of necessity — the tooling in this space either didn't exist or was locked behind paywalls. Open source only works if people give back.

## Pull Requests

1. Fork and branch from `main`
2. Add tests if you add code
3. Update docs if you change APIs
4. Make sure tests pass: `npm run test --workspace=apps/extension`
5. Open a PR with a clear description

## Bug Reports

File an issue at https://github.com/astranero/magpie/issues with:
- What you were trying to do
- What happened instead
- Steps to reproduce
- Browser version and OS
